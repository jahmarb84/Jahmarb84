## 🛠️ Projects

### 🔐 LogSentinel — Log Parser & Threat Analyzer
> *Cybersecurity | Python | Flask | Threat Detection*

A full-stack web application that parses system and server logs and automatically detects security threats. Built from scratch with a dark-themed dashboard UI.

**What it detects:**
- 🔴 Brute force attacks & SSH password spraying
- 🔴 SQL injection & XSS attempts
- 🟠 Directory/port scanning (Nikto, sqlmap, Nmap fingerprinting)
- 🟠 Privilege escalation via sudo
- 🟡 DDoS-pattern high request rates
- 🟡 Multi-account credential stuffing from single IP

**Tech stack:** `Python` `Flask` `Regex` `HTML/CSS` `JavaScript`

**Supports:** Apache access logs · Linux auth.log · Windows Event Logs

📁 [View Repository](https://github.com/jahmarb84/log_analyzer)
